# Web Monitor

Web Monitor is a web application for monitoring website performance and
uptime. The application is meant as an example that cover basic functionality 
of Google App Engine.

## Products
- [App Engine][1]

## Language
- [Python][2] ver 2.7

## APIs and services
- [NDB datastore API][3]
- [Scheduled Tasks][4]
- [Task Queues][5]

## Dependencies
- [webapp2][6]
- [jinja2][7]
